# FinalProject-ImageClone

## 開發環境

* android studio 4
* java

## 第三方 Package

* OpenCV

## Android Library

* CameraX
* safeargs
* navigation
* WindowManager
* Glide

## 適用平台

* Android 6.0 (API level 23)

## 功能

相機拍三張不同曝光程度的相片，使用 Exposure  Fusion 的算法，將三張圖片還原成 HDR 圖片。
